# 07_DECISIONS

Registro de decisiones en formato ADR (Architecture/business Decision Record): contexto, decisión tomada, alternativas descartadas, consecuencias.

Ningún ADR activo se contradice sin proponer formalmente su reemplazo. No se sobrescribe un archivo `status: active` sin registrar el cambio aquí o subir su versión.
