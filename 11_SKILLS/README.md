# 11_SKILLS

Skills instalables (paquetes `.skill`) y skills en construcción de JaraLab.

## Contenido actual

- `brainstorm.skill` — skill empaquetada y funcional.
- `decision-lab/`, `architecture/`, `growth/`, `cfo/` — estructura mínima (README + instrucciones base), aún sin metodología completa. Ver cada carpeta para su estado.

## Convención

Una skill se empaqueta como `.skill` solo cuando su metodología está definida y probada. Antes de eso, vive como carpeta de trabajo con README + instrucciones-base.md.
