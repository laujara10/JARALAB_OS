# 06_OPERATIONS

Procedimientos operativos estándar (SOPs), playbooks y workflows repetibles, paso a paso.

Un playbook es conocimiento validado (nivel de confianza C2 o superior) empaquetado para ejecución — no una idea sin probar. Se formaliza aquí cuando un hallazgo de `08_RESEARCH` se convierte en una oferta o proceso replicable.
