# 09_DAILY_LOG

Bitácora cronológica de la operación. Un archivo por sesión de trabajo relevante, con prefijo de fecha `YYYY-MM-DD-`.

Es un registro histórico: no se reescribe para reflejar cambios posteriores de estructura o de nombre. Si algo cambió después, el cambio se documenta en una entrada nueva, no editando la vieja.
